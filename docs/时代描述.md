- **生存时代 (Survival Age):**
    - **定义:** **原始社会，石器时代晚期**。人类以**部落**为单位进行**狩猎和采集**。没有文字，知识全靠口耳相传。思考方式**极其具象**，只能理解物理世界的直接交互（砸、烧、推、拉）。所有发明都围绕着最基本的生存需求：**取暖、食物、安全**。
- **城邦时代 (City-State Age):**
    - **定义:** **农业革命后，青铜/铁器时代早期**。人类开始**定居**，形成村落和早期城市。**私有财产**的概念出现，导致了社会分工和简单的**物物交换**。使用**象形文字或符号**进行记事。思考方式开始包含**初步的社会规则和抽象概念**（所有权、价值）。
- **帝国时代 (Empire Age):**
    - **定义:** **古典时期，铁器时代**。强大的**中央集权国家**出现，通过**官僚体系和成文法**统治广袤的疆域。大规模工程（如**道路、长城**）成为可能。**民族和国家认同感**开始形成，历史以**史诗和官方记录**的形式被书写。思考方式宏大，注重**秩序、效率、征服和集体叙事**。
- **理性时代 (Rationality Age):**
    - **定义:** **帝国晚期至中世纪中期**。在物质扩张达到瓶颈后，文明转向内部探索。**哲学、逻辑学、数学和天文学**得到发展。知识被精英阶层（如学者、僧侣）集中在**图书馆或学院**中。出现了复杂的**机械装置（齿轮、杠杆）**，但尚未有统一的科学方法论。
- **信仰时代 (Faith Age):**
    - **定义:** **中世纪**。这是一个**宗教和神学**主导一切的时代。世界的解释权归于**教会和圣典**。思考方式高度**象征化和形而上**，艺术、建筑、音乐都为了服务于**“神”**。**教权**成为与君权并立甚至更高的权力。
- **启蒙时代 (Enlightenment Age):**
    - **定义:** **文艺复兴至第一次工业革命前**。**“人”的价值被重新发现。科学方法成为探索世界的关键工具，挑战宗教和君权的绝对权威。人权、民主、自由等思想开始萌芽和传播。对自然力的探索从敬畏转向理解和驾驭（如电力）**。
- **全球时代 (Global Age):**
    - **定义:** **工业革命至信息时代早期**。**蒸汽机、内燃机**等技术彻底改变了生产力。世界通过贸易、战争和殖民主义紧密连接成一个整体。**民族国家**间的竞争白热化。**信息（电报、电话、早期计算机）**的传播速度首次超越交通工具。
- **分野时代 (Divergence Age):**
    - **定义:** **近未来**。在解决了大部分物质和信息问题后，人类面临存在主义的抉择。**生物技术、人工智能和太空探索**成为前沿。文明的走向出现分歧：是向外探索宇宙（**宏观路线**），还是向内探索意识的奥秘（**微观路线**）。
- **星辰时代 (Stellar Age):**
    - **定义:** **远未来，太空文明**。人类已经成为跨行星物种。思考的尺度扩展到**星系和光年**。物理学的边界（**相对论、时空**）成为新的囚笼。文明的核心议题是**星际航行、地外殖民和宇宙尺度的资源利用**。
- **奇点时代 (Singularity Age):**
    - **定义:** **终极未来，后人类时代**。**人工智能超越人类智能**，**生物肉体和数字虚拟的界限模糊**。意识、记忆、身份都可以被数据化。思考方式进入我们目前无法理解的高维层面，核心议题是**“存在”本身的定义**。
