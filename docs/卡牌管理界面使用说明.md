# 卡牌管理界面使用说明

## 访问地址

```
http://43.161.234.121/cardsdatabase.html
```

## 功能说明

### 1. 统计信息
顶部显示三个统计卡片：
- **基础卡牌** - 系统预设的初始卡牌数量
- **用户生成** - 玩家合成创建的卡牌数量
- **总计** - 所有卡牌总数

### 2. 筛选功能
- **全部来源** - 显示所有卡牌
- **基础卡牌** - 只显示系统预设卡牌
- **用户生成** - 只显示玩家创建的卡牌
- **时代筛选** - 按时代筛选（生存时代、城邦时代等）

### 3. 创建新卡牌
点击 **"+ 创建新卡牌"** 按钮，填写以下信息：

#### 必填字段
- **名称** - 卡牌名称（必须唯一）
- **类型** - 卡牌类型（如：element, material, concept）
- **稀有度** - common, uncommon, rare, epic, legendary

#### 可选字段
- **卡牌类型** - 灵感卡/钥匙卡/奖励卡
- **时代** - 所属时代（可为空）
- **解锁条件** - 如何解锁该卡牌（如：寒冷, branch:order）
- **初始手牌** - 勾选表示是起始卡牌
- **迷惑卡** - 勾选表示是迷惑卡
- **描述** - 卡牌的描述信息

### 4. 编辑卡牌
1. 在卡牌列表中找到要编辑的卡牌
2. 点击 **"编辑"** 按钮
3. 修改信息
4. 点击 **"保存"** 按钮

### 5. 删除卡牌
1. 在卡牌列表中找到要删除的卡牌
2. 点击 **"删除"** 按钮
3. 确认删除操作

**注意：** 删除操作不可恢复，且如果卡牌已被使用将无法删除

## 常见问题

### Q: 为什么看不到某些卡牌？
A: 检查筛选条件，尝试选择"全部来源"和"全部时代"

### Q: 编辑后没有生效？
A: 刷新页面查看，确认是否有错误提示

### Q: 无法删除卡牌
A: 该卡牌可能已被玩家使用，系统会保护这些卡牌

### Q: 下拉框选项看不清
A: 已修复，现在选项是深色背景+白色文字

## 技术支持

- 前端：独立HTML页面
- 后端API：`/api/cards-database-public/`
- 认证方式：Nginx Basic Auth
- 数据库：PostgreSQL
